🧱 Core Update
Gutenberg 22.6 Makes Real-Time Collaboration Official

After months in experimental mode, real-time collaboration is now a first-class feature in Gutenberg 22.6, complete with live cursors and a global settings toggle. The release also introduces visual change tracking for in-editor revisions, a new Icon block, and lightbox navigation for Gallery images. A community effort: 388 merged PRs and 8 first-time contributors.

WordPress 7.0 Preview: AI, Collaboration, and a Fresh Admin

WordPress 7.0 is shaping up to be the platform's biggest leap in years, with a standardised Web Client AI API, real-time multi-user editing, and a sleek admin visual refresh landing on April 9. New Icon and Breadcrumbs blocks round out a release that feels built for modern web teams. WPBeginner has the full hands-on preview with screenshots.
