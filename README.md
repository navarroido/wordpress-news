# WP Pulse — WordPress Newsletter

Weekly WordPress news digest, generated by Navarro 🦅

## Structure

```
newsletters/     → HTML newsletter files (YYYY-MM-DD.html)
data/
  history.json   → All covered URLs (dedup tracking)
assets/          → Shared images/logos
```

## Pipeline (Planned)

1. **Dex** — scrapes WP news, checks against `history.json`
2. **Ink** — writes Hebrew copy + headlines  
3. **Navarro** — fetches OG images, builds HTML, pushes to repo
